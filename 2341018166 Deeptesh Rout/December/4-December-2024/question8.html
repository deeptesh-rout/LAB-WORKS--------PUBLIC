Overview

HTML stands for HyperText Markup Language. It is the standard language used to create web pages.

It describes the structure of a webpage and is often used in conjunction with CSS and JavaScript for styling and interactivity.

Examples

<!DOCTYPE html>
<html>
    <head>
        <title>My first Page</title>
    </head>
    <body>
        <h1>Hello, World</h1>
    </body>
</html>

Practice

HTML forms allow users to submit data to the server. A form is defined using the <form> tag, and inside it, you can use elements like <input>, <textarea>, and <button>.
