Color Classes

Whenever you want to convey a specific meaning, use the following classes:

If text-muted is used, then the text appears muted, usually grayed out.

If text-primary is used, then the text appears blue. , indicating information

If text-success is used, then the text appears green , indicating success

If text-info is used, then the text appears light blue , indicating success.

If text-warning is used, then the text appears yellow

If text-danger is used, then the text appears red

If text-secondary is used, then the text appears grey

If text-dark is used, then the text appears green

If text-body is used, then the text appears green

If text-light is used, then the text appears black

If text-white is used, then the text appears black.
